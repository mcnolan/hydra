Hydra The Open Source writing project

What is the Hydra Project?

The Hydra project is an open source, collaborative writing project. We provide the tools to read and write books that have been written by one or more authors.

How can I participate?

You can choose to contribute in any way you like. Start a new book, add content to an existing book - even edit existing chapters. Don't like someone else's contribution? Provide an alternative!

Can I help improve the website?
